# Omarchy Hex Theme

A neon-leaning, dark theme with hex-magenta and cyan accents, tuned for Omarchy.

## Install

Use the Omarchy theme installer:

```
omarchy-theme-install https://github.com/OldJobobo/omarchy-hex-theme
```

## Palette

- Background: `#241b30`
- Foreground: `#f2f2e3`
- Accent magenta: `#ff00f6`
- Accent cyan: `#00fbfd`
- Blue: `#6e29ad`
- Red: `#e60a70`
- Green: `#00986c`
- Yellow: `#adad3e`
